# Portofolio Saya
Projek ini untuk kebutuhan portofolio saya. Silahkan bagi kalian yang membutuhkan referensi web portofolio bisa clone projek saya.

# Font 
- Inter - Untuk body text  
- Roboto Mono - Untuk headings dan code-like elements

# Deploy
1. Clone/download repository
2. Edit file index.html untuk mengubah konten
3. Ganti gambar di folder images/ dengan foto Anda
4. Sesuaikan warna dan styling di style.css
5. Deploy ke hosting pilihan Anda

# Lisensi
Portofolio ini tersedia untuk penggunaan pribadi dan komersial.
